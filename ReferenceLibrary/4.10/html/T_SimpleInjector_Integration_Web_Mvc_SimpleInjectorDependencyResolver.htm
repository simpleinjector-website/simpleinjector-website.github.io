Examples
 

             The following example shows the usage of the SimpleInjectorDependencyResolver in an
             MVC application:
             
 
C#
 
Copy
public class MvcApplication : System.Web.HttpApplication
{
    protected void Application_Start()
    {
        var container = new Container();

        // Make the container registrations, example:
        // container.Register<IUserRepository, SqlUserRepository>();

        container.RegisterMvcControllers(Assembly.GetExecutingAssembly());
        container.RegisterMvcIntegratedFilterProvider();

        // Create a new SimpleInjectorDependencyResolver that wraps the,
        // container, and register that resolver in MVC.
        System.Web.Mvc.DependencyResolver.SetResolver(
            new SimpleInjectorDependencyResolver(container));

        // Normal MVC stuff here
        AreaRegistration.RegisterAllAreas();

        RegisterGlobalFilters(GlobalFilters.Filters);
        RegisterRoutes(RouteTable.Routes);
    }
}

             The previous example show the use of the
             RegisterMvcControllers and
             RegisterMvcIntegratedFilterProvider
             extension methods and how the SimpleInjectorDependencyResolver can be used to set the created
             Container instance as default dependency resolver in MVC.
