CreateTService(IEnumerableType)
Creates a collection of serviceTypes, whose instances will be resolved lazily each time the returned collection of TService is enumerated. The underlying collection is a stream that will return individual instances based on their specific registered lifestyle, for each call to Current. The order in which the types appear in the collection is the exact same order that the items were supplied to this method, i.e the resolved collection is deterministic.
