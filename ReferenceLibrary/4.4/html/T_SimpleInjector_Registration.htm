Remarks
 
Lifestyle implementations create a new Registration instance for each registered
            service type. Expressions returned from the 
            BuildExpression method can be 
            intercepted by any event registered with ExpressionBuilding, have 
            initializers 
            applied, and the caching particular to its lifestyle have been applied. Interception using the 
            Container.ExpressionBuilt will not 
            be applied in the Registration, but will be applied in InstanceProducer.
