RegisterLifetimeScopeTConcrete(Container) Obsolete.
Registers that a single instance of TConcrete will be returned for each lifetime scope that has been started using BeginLifetimeScope. When the lifetime scope is disposed and TConcrete implements IDisposable, the cached instance will be disposed as well. Scopes can be nested, and each scope gets its own instance.
