SimpleInjectorWebExtensionsRegisterPerWebRequestTService Method (Container, FuncTService, Boolean)Simple Injector

Note: This API is now obsolete.

Registers the specified delegate that allows returning instances of TService and the returned instance will be reused for the duration of a single web request and ensures that, if the returned instance implements IDisposable, and disposeInstanceWhenWebRequestEnds is set to true, that instance will get disposed on the end of the web request.
