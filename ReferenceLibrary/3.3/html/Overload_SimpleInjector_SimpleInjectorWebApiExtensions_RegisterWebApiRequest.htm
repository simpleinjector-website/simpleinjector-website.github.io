RegisterWebApiRequestTConcrete(Container, Boolean) Obsolete.
Registers that a single instance of TConcrete will be returned for each Web API request. When the Web API request ends, and TConcrete implements IDisposable, the cached instance will be disposed. Scopes can be nested, and each scope gets its own instance.
RegisterWebApiRequestTService, TImplementation(Container, Boolean) Obsolete.
Registers that a single instance of TImplementation will be returned for the duration of a single Web API request. When the Web API request ends, disposeWhenScopeEnds is set to true, and the cached instance implements IDisposable, that cached instance will be disposed.
RegisterWebApiRequestTService(Container, FuncTService) Obsolete.
Registers the specified delegate that allows returning instances of TService, and returned instances are cached during the lifetime of a Web API request. When the Web API request ends, and the cached instance implements IDisposable, that cached instance will be disposed.

Registers the specified delegate that allows returning instances of TService, and returned instances are cached during the lifetime of single Web API request. When the Web API request ends, disposeWhenScopeEnds is set to true, and the cached instance implements IDisposable, that cached instance will be disposed.
