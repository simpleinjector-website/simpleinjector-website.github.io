RegisterPerWcfOperationTConcrete(Container) Obsolete.
Registers that a single instance of TConcrete will be returned during the WCF configured lifetime of the WCF service class. When the WCF service class is released by WCF and TConcrete implements IDisposable, the cached instance will be disposed as well.
RegisterPerWcfOperationTService, TImplementation(Container) Obsolete.
Registers that a single instance of TImplementation will be returned during the WCF configured lifetime of the WCF service class. When the WCF service class is released by WCF and TImplementation implements IDisposable, the cached instance will be disposed as well.
RegisterPerWcfOperationTService(Container, FuncTService) Obsolete.
Registers the specified delegate that allows returning instances of TService, and returned instances are cached during the WCF configured lifetime of the WCF service class. When the WCF service class is released by WCF and the cached instance implements IDisposable, that cached instance will be disposed as well.
RegisterPerWcfOperationTService(Container, FuncTService, Boolean) Obsolete.
Registers the specified delegate that allows returning instances of TService, and returned instances are cached during the WCF configured lifetime of the WCF service class. When the WCF service class is released by WCF, disposeWhenRequestEnds is set to true, and the cached instance implements IDisposable, that cached instance will be disposed as well.
