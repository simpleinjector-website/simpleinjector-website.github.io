  NameDescription
RegisterPerWebRequestTConcrete(Container) Obsolete.
Registers that one instance of TConcrete will be returned for every web request and ensures that -if TConcrete implements IDisposable- this instance will get disposed on the end of the web request.
RegisterPerWebRequestTService, TImplementation(Container) Obsolete.
Registers that one instance of TImplementation will be returned for every web request every time a TService is requested and ensures that -if TImplementation implements IDisposable- this instance will get disposed on the end of the web request.
RegisterPerWebRequestTService(Container, FuncTService) Obsolete.
Registers the specified delegate that allows returning instances of TService and the returned instance will be reused for the duration of a single web request and ensures that, if the returned instance implements IDisposable, that instance will get disposed on the end of the web request.
RegisterPerWebRequestTService(Container, FuncTService, Boolean) Obsolete.
Registers the specified delegate that allows returning instances of TService and the returned instance will be reused for the duration of a single web request and ensures that, if the returned instance implements IDisposable, and disposeInstanceWhenWebRequestEnds is set to true, that instance will get disposed on the end of the web request.
