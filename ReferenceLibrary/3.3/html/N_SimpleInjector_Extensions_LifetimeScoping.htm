SimpleInjector.Extensions.LifetimeScoping NamespaceSimple Injector
The SimpleInjector.Extensions.LifetimeScoping namespace is part of the SimpleInjector.Extensions.LifetimeScoping.dll and contains functionality for registering services using the LifetimeScopeLifestyle. This lifestyle alows users to explicitly begin and end a scope on a single thread. Within that scope, a single instance of the registered service will be returned.
Classes
  ClassDescription
LifetimeScopeLifestyle
Defines a lifestyle that caches instances during the lifetime of an explicitly defined scope using the BeginLifetimeScope method. A scope is thread-specific, each thread should define its own scope. Scopes can be nested and nested scopes will get their own instance. Instances created by this lifestyle can be disposed when the created scope gets disposed.
