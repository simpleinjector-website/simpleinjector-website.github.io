BeginLifetimeScope

             Begins a new lifetime scope for the given container on the current thread. 
             Services, registered with 
             RegisterLifetimeScope or
             using the LifetimeScopeLifestyle and are requested within the same thread as where the 
             lifetime scope is created, are cached during the lifetime of that scope.
             The scope should be disposed explicitly when the scope ends.

RegisterLifetimeScopeTConcrete(Container) Obsolete. 

            Registers that a single instance of TConcrete will be returned for
            each lifetime scope that has been started using 
            BeginLifetimeScope. When the 
            lifetime scope is disposed and TConcrete implements IDisposable,
            the cached instance will be disposed as well.
            Scopes can be nested, and each scope gets its own instance.
